# 概要
これは、CANELÉ / CROISSANT﻿ WAG(徳島にあるカヌレ専門店)のホームページです。

**☆ポイント☆**

**・ベージュ系の配色に統一し、お店の雰囲気に合わせた。**

**・お店のロゴである犬を引き立たせるために、ローディング画面に挿入し、動きをつけて犬が走っているようなイメージをつけた**

<img width="1111" alt="スクリーンショット 2024-01-10 140322" src="https://github.com/ayukasaijo1103/wag/assets/135251771/42346f82-ee1b-4d2b-bd54-3daf07815f3f">


**・商品が目を惹くHPにするために、商品画像を中央に大きく設置し、スライドの動きをつけてさまざまなメニューの認知を図った。**

<img width="1114" alt="スクリーンショット 2024-01-10 140719" src="https://github.com/ayukasaijo1103/wag/assets/135251771/6eed11f9-6dde-4502-b546-58473f9317d8">


**・店主の想いを伝えるために、店名の由来を記載し、印象に残るように下方向から柔らかく表示されるアニメーションをいれた。**

<img width="1112" alt="スクリーンショット 2024-01-10 140752" src="https://github.com/ayukasaijo1103/wag/assets/135251771/362c53e8-ddd8-4362-8806-459bdaccc2d3">


**・インスタグラムの集客につなげるためNEWSページを作成。インスタグラムの投稿とリンクさせている。**

<img width="1103" alt="スクリーンショット 2024-01-10 140841" src="https://github.com/ayukasaijo1103/wag/assets/135251771/34d5cd37-ce11-4dcb-9e2f-2ff331ee8ddb">


**・何の商品がどれくらいの種類あるのか分かりやすくるためにスライドショーで各メニューを紹介**

<img width="1108" alt="スクリーンショット 2024-01-10 140924" src="https://github.com/ayukasaijo1103/wag/assets/135251771/a56699bd-e967-4ecc-b887-92786bb8d8f3">


**・地方にある小さなお店だからこそ、お客様との距離を近づけるためにCONTACTページを設置。G-mailリンクと紐づけている。**
